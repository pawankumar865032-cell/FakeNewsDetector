# Fake News Detection Using Machine Learning

## Overview
This project classifies news articles as Fake or Real using NLP and Machine Learning.

## Features
- News classification
- Confidence score
- Streamlit web interface

## Technologies
- Python
- NLP
- TF-IDF
- Logistic Regression
- Streamlit

## Dataset
Fake and Real News Dataset (Kaggle)

## Installation

pip install -r requirements.txt

streamlit run app.py
